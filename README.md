synchro-seedbox-ng
===============

Installation :
===============
cd /tmp

git clone https://github.com/ag0r4n/synchro-seedbox-ng

cd synchro-seedbox

chmod +x makeinstall.sh

./makeinstall.sh


Configuration :
================

Configuration file is /usr/local/etc/synchro-seedbox-ng/synchro.conf

Usage :
========

Execute SSNG command

# ssng /my/seedbox/folder /my/nas/folder
